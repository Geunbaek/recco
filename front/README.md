- [x] 카테고리 대분류와 소분류를 선택하여 상품을 보여준다.

- [x] 검색결과의 상품, 브랜드, 성분 탭별 상품을 보여준다.
- [x] 검색결과를 최신순, 이름순, 가격순으로 보여준다.

- [x] 5개 이상의 컨텐츠를 선택하고 결과를 확인하기를 누르면 추천 결과 페이지로 넘어간다.
